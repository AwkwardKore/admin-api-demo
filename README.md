## Requirements
* Ruby version: 2.3.1
* Postgresql version: 9.5.3

## Setup
1. Install project gems: `$ bundle install`
2. Create file for environment variables: [config/application.yml](config/application.example.yml)
3. Create database: `$ rake db:create`
